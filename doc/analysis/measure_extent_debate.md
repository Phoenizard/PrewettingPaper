# extent 度量的论证记录（agent-team）

本文件记录 extent 度量（pre-wetting line 长度 + 到 binodal 距离）在 omega topic 上
的论证过程：提方案 → 抽数据+分析 → 隔离怀疑者审查 → 辩论收束。每轮一节，最新在上。

## 度量定义（当前，omega topic 敲定）

每 case 合并全部 pre-wetting 点为一个无序点集（不分 branch），在原始
$(\phi_{1,\infty}, \phi_{2,\infty})$ 平面（两轴同量纲，不归一化）给两个头条量：

- 长度 $L$：建欧氏最小生成树（MST），剪掉长度 $> \text{gap\_tol}=0.015$ 的边（跨簇桥
  边），$L=$ 剩余边长之和（= 各段弧长之和，空隙不计）。诊断量：段数 $n_{seg}$、被剪
  桥边总长 gap_total、不剪的全 MST 长 full_length。
- 到 binodal 距离：逐点到 binodal 最近欧氏距离；头条 $\bar d=$ 其平均（典型深入单相
  区的深度）。诊断量：$d_{\max}$、$d_{\min}$。

gap_tol=0.015 标定依据：单条扫描线内相邻点典型间距约 0.0014，远场网格步长 0.02；
0.015 约为线内间距 10 倍（不切碎真线）、又小于网格步长（能分开真正分离的簇）。

## Round 1（2026-07-12）

### 结果（omega topic：T-a，chibb=0，11×11 omega 网格 [-0.50,-0.30] 步长 0.02）

- 121 case，119 有 pre-wetting，2 熄灭（均在 omega1=-0.50 强吸引角）。
- 沿 omega1（对 omega2 平均）从 -0.50 到 -0.30（强→弱溶质 1 吸引）：$L$ 单调升
  0.010→0.081（约 8 倍）；$\bar d$ 单调降 0.0214→0.0038。二者反号。
- 沿 omega2：$L$ 仅缓升 0.027→0.053，$\bar d$ 基本持平（~0.012）。omega1 主控、
  omega2 近乎惰性。
- 物理读法：溶质 1 墙面吸引减弱（omega1 变正），pre-wetting line 变长、且更贴近
  binodal；吸引增强则线缩短（趋向强吸引角熄灭）、但在熄灭前伸入单相区更深。机制与
  angle1_omega.md 一致：过强吸引把一级 thin/thick 跳变推过临界点、变连续吸附、线消失。

### 隔离怀疑者审查（全新 agent，只喂物理+项目背景+结果，不含度量设计动机）

裁决：SOUND（度量定义无需改写），带 1 个 caveat + 2 个报告层修正。怀疑者独立在真实
点云上跑了反驳性检验：

- $L$ 不是在追点数/分辨率：对一条真线 2×/4×/8× 降采样，$L$ 变动 <2%（MST 弧长密度不
  变）。近直的 omega 线上 $L=$ pca_span = bbox 对角，四位一致。
- $L$ 未被扫描窗 $\phi_{\max}=0.2$ 截断：真线终止在 $\phi\approx0.09\text{–}0.11$，
  远在窗内，线终于其临界点、非数值墙。
- $L$、$\bar d$ 的反相关是真物理：单条线内到 binodal 距离沿线单调增（近临界端 0.0018
  →远端 0.0047），故长线本应把 $\bar d$ 拉高——而观测是长线 $\bar d$ 更小，几何耦合
  与观测趋势反向，说明真物理（整条线随 omega1 变弱而整体向 binodal 迁移）强到压过几何
  耦合。这反证 $L$ 与 $\bar d$ 确是两个独立维度。
- $\bar d$ 不是离散化假象：binodal 近线处局部间距 0.0006（CSV 里 0.012 的大间距是拼接
  不相连分支所致）；binodal 加密 40× 后 $\bar d$ 仅变 6%。
- gap_tol 未制造/掩盖趋势：真线上 gap_tol∈{None,0.05,0.015,0.005} 时 $L$ 不变，到
  0.002 才开始切；0.015 稳在平台。

### 辩论与验证（对怀疑者 3 条修正逐条处理）

1. 报告 closest-approach 用 $d_{\min}$，不要只用 $\bar d$。采纳——「线离 binodal 多
   近」是最近接近问题，$\bar d$ 被远端尾部拉高、不宜兼任；$d_{\min}$ 已在表中，明确
   分工：$\bar d$=典型深度、$d_{\min}$=最近贴合。度量定义不改，只是报告时并列 $d_{\min}$。

2. 近熄灭区 $L$ 低置信 + gap_tol/网格 cliff 隐患。已在真实数据核实：
   - 2 个「熄灭」cell 的原始 pre-wetting 点数确为 0——是真熄灭，非 gap_tol 误杀。
   - 沿 omega1=-0.50 逼近熄灭：$L$ 平滑降 0.0149→0.0095，随后 1 点 cell（$L=0$，
     单点本无长度）再到真空，非断崖；该列 $L_{0.015}=L_{\text{no-cut}}$ 处处相等，
     gap_tol 未切任何边。
   - 全 omega topic 仅 1 个 case 满足「$n\ge2$ 且 $L=0$」（两点恰好隔一个网格步 0.02
     被剪）——孤例、近乎测度零，非系统性 cliff。
   结论：怀疑者的 cliff 隐患原理上成立、但在本数据里可忽略。报告时对近熄灭区 $L$ 交叉
   核 $n_{points}$/$n_{seg}$，不让「$n\ge2$ 且 $L=0$」的孤例被读成熄灭即可。

3. 说明 $\bar d$ 用的 binodal 是近线密采样。采纳为报告注记：当前 min_dist_to_set 直接
   对原始 binodal CSV，在本 case 因近线处密而无碍；论文里注明距离对的是近线密采样分支，
   保证可复现。度量代码不改。

### 收束

怀疑者裁决 SOUND；3 条修正均为报告层（并列 $d_{\min}$、近熄灭 $L$ 交叉核、注明
binodal 密采样），不触及度量定义；cliff 隐患经真实数据核实可忽略。本轮无「触及定义」
的未决质疑 → Round 1 收束为「度量定义成立」。
